Fatal Accidents in United States - Year 2015

About The Visualization Fatal Accidents by State Highways vs Other Roads <Prev Next>

The Project

The goal of this project is to assist the readers in visualizing the number of fatal accidents that occurred in the United States of America in the year 2015. The project also provides detailed information on the number of fatal accidents that occurred on highways and on other roads.The narrative visualization follows the interactive slide show structure. The first scene presents the general information about the project, the dataset used in the project, the visualization structure, its elements and the technical references. The second and third scene are more interactive and allow readers to get to know about the number of fatal accidents by state and the number of fatal accidents on highways vs local roads. Readers can select the state from the drop down list and the selected state will be highlighted for reference.

Scenes

The first scene provides an overview on the total number of fatal accidents by state. The readers will be seeing a bubble chart with each bubble representing a state and the size of it corresponding to the total number of accidents in the state. The reader can get further information by hovering over the bubbles. This will produce a tooltip which will show the full state name and the number of fatal accidents.

The second scene provides information on the number of fatal accidents that occurred on highways in comparison to the other local roads. A stacked bar chart was used to depict this scene. The readers can hover over the stacked bars and gather more information though the tooltip.

All the scenes are split into two panel or columns. The first scene is split is split equally and both columns have equal space. The rest of the scenes are split in the ratio of 75:25, and the left columns are used for the charts and the right columns are used for the text description.

Annotations

The scenes contain static annotations that highlight some interesting details seen on the charts and are cleared between the scenes. In addition to these the scenes also contain tooltips that provide further information as users hover over each the different areas in the chart.

Parameters

The main parameter is the scene name that is displayed. This parameter drives the gathering of data that is needed for the scenes as well as the annotations that are displayed for the scene.

Triggers

The trigger is the selection of the scene name by the reader. This triggers the loading of the required data and generating the charts, the annotations and the text description displayed on the right side of the webpage.

Dataset

The dataset used here is a public data available on the link- https://www.kaggle.com/usdot/nhtsa-traffic-fatalities

The dataset is published by National Highway Traffic Safety Administration (NHTSA). The NHTSA's Fatality Analysis Reporting System(FARS) publishes an annual report for the fatal accidents that occurred over last year. The data set has around 70 columns and custom python scripts were used to generate the data needed for this project in csv files.
